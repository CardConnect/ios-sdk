Inherits from NSObject
Conforms to UITextFieldDelegate

Overview

A custom delegate proxy that conforms to UITextFieldDelegate.

This is the base class used in CCCCardFormatterDelegate, CCCCVVFormatterDelegate and CCCExpirationDateFormatterDelegate.
