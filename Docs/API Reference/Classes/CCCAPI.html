Overview

This class is used to make the API call to generate a token for a card object.

Its a singleton class and endpoint should be set before calling generateAccountForCard:completion:. The endpoint will be provided to you by CardConnect. Logging on the api call can be enabled by setting enableLogging to TRUE.

Examples:

  • [CCCAPI instance].endpoint = @“{your card secure endpoint}”;
  • [[CCCAPI instance] generateAccountForCard:card completion:];
