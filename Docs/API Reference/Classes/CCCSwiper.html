Inherits from NSObject
Declared in CCCSwiper.h

Overview

CCCSwiper is a base class that defines a swiper controller. To use the swiper, please use CCCSwiperController. If the Library doesn’t contain CCCSwiperController, this version does not support swiper integration.

shouldShowAudioWarning

Indicates if an audio warning should be displayed.

- (bool)shouldShowAudioWarning

Return Value

A flag for if an audio warning should be displayed.

Discussion

If returns TRUE, this swiper utilizes the audio jack for communication. An audio warning should be shown prior to initializing the swiper that indicates the user should remove any headphones and turn the volume to max.
