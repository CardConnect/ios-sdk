swiper:didFailWithError:completion:

A required method notifying the delegate that CCCSwiper failed to swipe the card.

- (void)swiper:(CCCSwiper *)swiper didFailWithError:(NSError *)error completion:(void ( ^ ) ( ))completion

Parameters

swiper

An instance of CCCSwiper.

error

The error that occurred if available.

completion

Block that should be called to start the swiper again.

Discussion

Call completion to start the swiper again. If you presented a status indicator or disabled user interaction in [CCCSwiperDelegate swiperDidStartMSR:], dismiss the status indicator and re-enable user interaction at this point before calling completion.
